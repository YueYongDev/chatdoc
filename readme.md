# ChatDoc

一个使用 `zhipuAI` 与文档对话的例子, 目前支持的文档类型 `.pdf`, `.epub`, `.md`, `.txt`, `.docx`, `web`

## 后端

后端为私有项目 👉
[chatdoc-backend](https://github.com/YueYongDev/chatdoc-backend)

## 启动前端

```shell
# my local node version v16.11.0
cd client
yarn
npm run dev
```
